## Установка

Отдельно от всей библиотеки можно скачать и подключить только переменные.

```bash
yarn add @savosya/savosya-myuikit-variables
```

Далее переменные можно подключить как обычный CSS файл.

```jsx
import '@savosya/savosya-myuikit-variables/index.css'
// или
import '@savosya/savosya-myuikit-variables/colors.css'
```
