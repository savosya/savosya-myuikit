# Типографика

#### Используемый шрифт ``Inter``

`--eub-font-family: 'Inter', system-ui, -apple-system, 'Segoe UI', Roboto, 'Helvetica Neue', Helvetica, sans-serif;`

##### Чтобы подключить шрифт скачайте его и подключите:

```css
@font-face {
    font-family: 'Inter';
    src: url('/fonts/Inter-Regular.ttf') format('truetype');
    font-weight: 400;
    font-style: normal;
}
@font-face {
    font-family: 'Inter';
    src: url('/fonts/Inter-Medium.ttf') format('truetype');
    font-weight: 500;
    font-style: normal;
}
@font-face {
    font-family: 'Inter';
    src: url('/fonts/Inter-Bold.ttf') format('truetype');
    font-weight: 700;
    font-style: normal;
}
```

или

```css
@import url('https://fonts.googleapis.com/css?family=Inter:400,500&display=swap');
```

В случае если у вас используется другой шрифт, есть возможность заменить шрифт в ките через `css` переменную `--eub-font-family`.
<br/>
Следует определить ее как можно выше

## Компонент Typography

#### Компонент предоставляет набор пресетов с определенным набором стилей для заголовком и текста.

##### Использование:

```jsx
import { Typography } from '@savosya/savosya-myuikit-root';
// или
import { Typography } from '@savosya/savosya-myuikit-typography';


const { Title, Text } = Typography
```
