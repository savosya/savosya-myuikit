# UI-KIT библиотека для разработки на React

# Установка

Установить всю библиотеку:

```bash
yarn add @savosya/savosya-myuikit-root
```

Каждый компонент публикуется отдельным пакетом, поэтому вы можете подключить только нужный, не устанавливая библиотеку целиком:

```bash
yarn add @savosya/savosya-myuikit-button
```

## Использование

```jsx
import { Button } from '@savosya/savosya-myuikit-button';
```
